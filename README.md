# Usage

* インストール設定/方法は[こちら](https://github.com/DensoITLab/marp-lab-theme/blob/main/howto.md)

* 細かい機能等は[example.md](https://github.com/DensoITLab/marp-lab-theme/blob/main/example.md)をmarp環境で参照
